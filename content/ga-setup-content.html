GA4 가입 및 설치

웹사이트에 GA4를 올바르게 설치하고 구성하는 방법

GA4 계정 생성

Google Analytics4를 시작하려면 먼저 계정을 생성합니다.

1단계: Google Analytics4 계정 생성

  1. Google Analytics4 접속
  2. 측정 시작 → Google 계정 로그인
  3. 계정 이름 입력 및 데이터 공유 설정

2단계: 속성 생성

  1. 속성 이름, 시간대, 통화 설정

3단계: 비즈니스 정보

  1. 업종, 규모, 사용 목적 선택

데이터 스트림 설정

웹 데이터 스트림

  1. 웹 선택 → URL/스트림 이름 입력
  2. 향상된 측정 활성화
향상된 측정
자동 추적: 페이지뷰, 스크롤, 아웃바운드 클릭, 사이트 검색, 동영상, 파일 다운로드

태그 설치

Google Tag Manager (권장)
코드 수정 없이 관리, 디버깅/버전관리 제공
GTM 설치 가이드
직접 설치 (gtag.js)
빠른 설치, 외부 의존성 없음
플러그인 (CMS)
WordPress: Site Kit, MonsterInsights 등

설치 확인

실시간 보고서

  1. 실시간 메뉴에서 방문 데이터 확인

개발자 도구

  1. Network에서 gtag/analytics 요청 확인
  2. Console 오류 확인
주의
실시간 반영까지 지연 가능. 개발 환경과 운영 환경을 구분하세요.
