GTM 가입 및 설치

Google Tag Manager 컨테이너 설정과 설치 방법

GTM 계정 생성

Google Tag Manager 계정을 생성하고 컨테이너를 설정하는 방법입니다.

1단계: GTM 계정 생성

  1. Google Tag Manager 접속
  2. Google 계정으로 로그인계정 이름 입력 (예: "회사명")
  3. 국가 선택

2단계: 컨테이너 생성

  1. 컨테이너 이름 입력 (예: "메인 웹사이트")
  2. 타겟 플랫폼 선택 (웹, iOS, Android)
  3. 컨테이너 생성 완료

GTM 코드 설치

생성된 GTM 컨테이너 코드를 웹사이트에 설치합니다.

설치 코드

📝 GTM 코드
1. Head 섹션
<script>
(function(w,d,s,l,i){w[l]=w[l]||[];w[l].push({'gtm.start':
new Date().getTime(),event:'gtm.js'});var f=d.getElementsByTagName(s)[0],
j=d.createElement(s),dl=l!='dataLayer'?'&l='+l:'';j.async=true;j.src=
'https://www.googletagmanager.com/gtm.js?id='+i+dl;f.parentNode.insertBefore(j,f);
})(window,document,'script','dataLayer','GTM-XXXXXXX');
</script>

설치 확인

GTM이 올바르게 설치되었는지 확인하는 방법입니다.

확인 방법

  1. 웹사이트 방문 후 F12로 개발자 도구 열기
  2. Console에서 "Google Tag Manager" 메시지 확인
  3. Network 탭에서 gtm.js 요청 확인
  4. GTM 미리보기 모드로 테스트
🔍 GTM 미리보기
GTM 인터페이스에서 "미리보기" 버튼을 클릭하면 실시간으로 태그 동작을 확인할 수 있습니다.
