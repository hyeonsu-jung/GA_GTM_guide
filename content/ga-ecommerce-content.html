전자상거래 태그

온라인 쇼핑몰의 매출과 구매 전환을 정확하게 추적

GA4 Enhanced Ecommerce

GA4의 Enhanced Ecommerce를 통해 온라인 쇼핑몰의 전체 구매 과정을 추적할 수 있습니다.

주요 추적 단계

  • 상품 조회: 상품 페이지 방문
  • 장바구니 추가: 상품을 장바구니에 추가
  • 장바구니 조회: 장바구니 페이지 방문
  • 결제 시작: 결제 프로세스 시작
  • 구매 완료: 결제 완료 및 주문 확정
💰 비즈니스 가치
Enhanced Ecommerce를 통해 매출, 전환율, 상품별 성과, 사용자 구매 패턴 등을 상세하게 분석할 수 있습니다.

상품 조회 추적

사용자가 상품 페이지를 방문할 때 상품 정보를 GA4로 전송합니다.

view_item 이벤트

📊 구현 예시
gtag('event', 'view_item', {
  'currency': 'KRW',
  'value': 50000,
  'items': [{
    'item_id': 'SKU123',
    'item_name': '무선 이어폰',
    'category': '전자제품',
    'quantity': 1,
    'price': 50000
  }]
});

필수 매개변수

  • item_id: 상품 고유 ID
  • item_name: 상품명
  • price: 상품 가격
  • currency: 통화 코드

장바구니 추적

장바구니 추가/제거와 장바구니 조회를 추적합니다.

🛒 add_to_cart
상품을 장바구니에 추가할 때 발생하는 이벤트입니다.
❌ remove_from_cart
장바구니에서 상품을 제거할 때 발생하는 이벤트입니다.
👀 view_cart
장바구니 페이지를 조회할 때 발생하는 이벤트입니다.

장바구니 추가 구현

💻 코드 예시
gtag('event', 'add_to_cart', {
  'currency': 'KRW',
  'value': 50000,
  'items': [{
    'item_id': 'SKU123',
    'item_name': '무선 이어폰',
    'category': '전자제품',
    'quantity': 1,
    'price': 50000
  }]
});

결제 프로세스 추적

결제 시작부터 완료까지의 과정을 단계별로 추적합니다.

결제 단계

  1. begin_checkout: 결제 프로세스 시작
  2. add_payment_info: 결제 정보 입력
  3. add_shipping_info: 배송 정보 입력
  4. purchase: 구매 완료
🔄 결제 시작
gtag('event', 'begin_checkout', {
  'currency': 'KRW',
  'value': 50000,
  'items': [{
    'item_id': 'SKU123',
    'item_name': '무선 이어폰',
    'category': '전자제품',
    'quantity': 1,
    'price': 50000
  }]
});

구매 완료 추적

가장 중요한 purchase 이벤트로 매출을 정확하게 측정합니다.

purchase 이벤트

✅ 구매 완료
gtag('event', 'purchase', {
  'transaction_id': 'T12345',
  'currency': 'KRW',
  'value': 50000,
  'tax': 5000,
  'shipping': 3000,
  'items': [{
    'item_id': 'SKU123',
    'item_name': '무선 이어폰',
    'category': '전자제품',
    'quantity': 1,
    'price': 50000
  }]
});

중요 매개변수

  • transaction_id: 주문 고유 번호 (중복 방지)
  • value: 총 구매 금액
  • tax: 세금
  • shipping: 배송비

전자상거래 보고서

GA4에서 전자상거래 데이터를 확인하고 분석하는 방법입니다.

주요 보고서

  • 수익 보고서: 매출, 전환율, 평균 주문 가치
  • 상품 성과: 인기 상품, 상품별 매출
  • 구매 여정: 구매 전환 경로 분석
  • 장바구니 분석: 장바구니 이탈률, 전환율
